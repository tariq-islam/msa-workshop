# Clean Up

Let's take a second to clean everything up.

### Clear out your project
```markdown

oc delete all --all --now
```

### Halt or Destroy your CDK instance
```markdown

# Halt for later work
vagrant Halt

# Destroy to start fresh
vagrant Destroy
```
